# Pensum V2 – neste steg for produktinnhold

Denne runden gjør Pensum Løsninger til en faktisk innholdskilde for rapportmotoren.

## Det som nå kan vedlikeholdes per produkt
- rolle i porteføljen
- benchmark
- risikonivå
- likviditet
- forventet avkastning
- forventet yield
- kort pitch
- investeringscase
- hvorfor produktet er inkludert
- nøkkelrisiko
- foretrukket diagram
- antall produktslides

## Anbefalt månedlig arbeidsflyt
1. Oppdater avkastningsdata via CSV/Excel
2. Oppdater eksponeringsdata ved behov
3. Gå til Pensum Løsninger admin og vedlikehold pitch/case/benchmark hvis noe har endret seg
4. Bruk v2-preview for å kontrollere hvilke produkter som havner i rapporten
5. Generer PowerPoint v2

## Det som bør bygges videre
- direkte lagring av region-/sektorbeskrivelser per produkt
- egen Morningstar-import per produkt
- valgfri LLM-generert tekst med strenge guardrails
- strengere slide-regler per produktkategori
